# E-Commerce Website Enhancement (Intermediate Task)

## Project Objective
This project enhances the UX of an e-commerce website by implementing:
- Advanced product filtering
- Sorting mechanisms
- Streamlined checkout flow
- User feedback collection
- Clean and responsive interface

## Features
- Search products instantly
- Filter by category
- Sort by price and rating
- Price range slider
- Add to cart / quantity update / remove item
- Multi-step checkout process
- Feedback box for usability testing insights

## Tech Stack
- React.js
- JavaScript
- CSS3
- Framer Motion
- Lucide React Icons

## Run Locally
```bash
npm install
npm start
```

## Folder Structure
- `src/App.js` → Main UI logic
- `src/data.js` → Product data
- `src/styles.css` → Styling
- `public/index.html` → Root HTML

## Suggested Viva / Report Points
1. Improved product discovery using search, filter, and sort.
2. Reduced checkout friction using fewer steps.
3. Better mobile responsiveness.
4. User feedback form supports iterative UX improvement.
